# FireUp

FireUp use puppet to deploy ur ruby/rails code to VPS.

It contains `postgresql`, `redis` and `nginx` recipes.

## How to use FireUp

### Vagrant

```bash
git clone git://github.com/SaitoWu/fireup.git
cd fireup
gem install librarian
librarian-chef init
vagrant up
```
